# 🚀 Complex and Dynamic Graph Systems & Algorithms (Java)

A curated collection of **advanced dynamic graph algorithms** implemented in Java, designed for **high-performance**, **real-time updates**, and **efficient query handling** in evolving graph structures.

---

## 📌 Overview

This repository focuses on solving complex problems where graphs are **not static** — edges and nodes can be added, removed, or modified over time.

It includes implementations of:
- Incremental Graph Algorithms  
- Decremental Graph Algorithms  
- Fully Dynamic Graph Algorithms  
- Offline & Online Query Processing Techniques  

---

## ⚙️ Algorithms Included

### 🔹 DSU with Rollback
- Supports undo operations on union-find  
- Useful for offline queries and divide & conquer  

### 🔹 Dynamic Transitive Closure
- Maintains reachability in a changing graph  
- Efficient for query-heavy problems  

### 🔹 Holm–de Lichtenberg–Thorup Algorithm
- Advanced fully dynamic connectivity algorithm  
- Handles edge insertions & deletions efficiently  

### 🔹 Offline Dynamic Connectivity
- Processes queries in batches  
- Uses segment trees + DSU rollback  

### 🔹 Online Dynamic MST
- Maintains Minimum Spanning Tree dynamically  
- Handles updates in real-time
